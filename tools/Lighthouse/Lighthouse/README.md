# Lighthouse

Electron + React + TypeScript desktop app with SQLite and YouTube Data API integration.

## Features
- Local SQLite database (better-sqlite3)
- YouTube Data API integration
- Dashboard UI
- Scheduled scans
- Case classification
- Weekly report generation

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```
2. Start the app in development mode:
   ```
   npm start
   ```
3. Build for production:
   ```
   npm run build
   ```
